## ❓ Usage (Windows)
1. Download Node.js v18.18.2:
- [Node.js v18.18.2 (64-bit)](https://nodejs.org/dist/v18.18.2/node-v18.18.2-x64.msi)
2. Install Node.js packages:
``` 
npm install 
```
3. Fill the requirements in config.json
4. Start the bot using run.bat
